# MarkMath

基于浏览器的积木式知识卡片编辑器。支持卡包管理、多卡片翻页、多种内容块组合、富文本编辑、双面卡片、自动保存等功能。

## 功能特性

### 卡包管理
- **卡包** — 首页以卡包为单位管理，每个卡包内包含多张卡片
- **卡片命名** — 顶部栏可为每张卡片独立命名
- **搜索** — 首页搜索框支持按卡片名称搜索，点击直接跳转
- **翻页切换** — 卡包内通过底部翻页栏切换不同卡片
- **自动保存** — 编辑内容后自动保存，无需手动操作

### 积木式编辑
- **文字块** — 富文本编辑，支持加粗、斜体、下划线、删除线、上下角标
- **图片块** — 本地图片上传与展示，支持替换图片、图片标注编辑、双面隐藏
- **音频块** — 本地音频上传与播放
- **选择题块** — 多选项编辑，标记正确答案
- **链接块** — URL 链接卡片

### 图片标注编辑器
- 基于 Excalidraw 的画布式图片编辑
- 支持矩形、椭圆、箭头、连线、文字、画笔等工具
- 工具锁定模式：选中工具后可连续绘制，无需反复切换
- 标注数据可保存/恢复，支持二次编辑
- 导出合成图片（标注 + 原图）

### 富文本工具栏
- 字体切换（楷体、黑体、宋体、圆体、手写、Serif、Mono）
- 字号调整（12px ~ 32px）
- 行间距调整
- 字体颜色 / 下划线颜色（右键选色）
- 自定义颜色持久化，常用色快捷选取
- LaTeX 公式插入（KaTeX 渲染）
- Emoji 表情面板（Twemoji 跨平台统一渲染）
- 文本对齐（左对齐 / 居中）
- 撤销操作（自定义 undo 栈）

### 卡片模式
- **双面隐藏** — 文字块/图片块可设为双面模式，预览时点击翻转
- **背面自定义颜色** — 双面背面支持从预设色板选择背景色，便于区分不同类型内容
- **挖空复习** — 已有挖空内容在预览时隐藏，点击逐个揭示

### 其他
- 拖拽排序（@dnd-kit）
- 块操作菜单 — 点击六点图标弹出菜单，支持删除、在前/后加块、颜色、复制、拷贝、粘贴
- 每个块可独立设置背景色（含自定义色持久化）
- 预览模式导出为 PNG 图片或 PDF（html-to-image + jsPDF，支持完整内容导出）
- 所有数据保存在 localStorage，刷新不丢失
- 旧版数据自动迁移为卡包格式

## 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | React 18 + TypeScript |
| 构建 | Vite 5 |
| 样式 | Tailwind CSS 3 |
| 拖拽 | @dnd-kit/core + sortable |
| 公式 | KaTeX |
| 截图 | html-to-image |
| PDF | jsPDF |
| 图片编辑 | Excalidraw（画布式标注） |
| Emoji | Twemoji（跨平台统一渲染） |
| 存储 | localStorage |

## 项目结构

```
src/
├── App.tsx                  # 应用主入口，卡包导航 + 翻页逻辑
├── types.ts                 # TypeScript 类型定义（CardPack, Card, Block）
├── store.ts                 # localStorage 持久化 + 数据迁移
├── index.css                # Tailwind + 自定义样式
├── components/
│   ├── CardList.tsx         # 卡包列表页（PackList）
│   ├── CardEditor.tsx       # 卡片编辑器（编辑/预览/导出）
│   ├── Toolbar.tsx          # 富文本格式工具栏
│   ├── BlockContainer.tsx   # 块容器（拖拽、操作菜单、背景色）
│   └── blocks/
│       ├── TextBlock.tsx    # 文字块（contentEditable）
│       ├── ImageBlock.tsx   # 图片块（支持双面）
│       ├── ImageCanvas.tsx  # 图片标注编辑器（Excalidraw）
│       ├── AudioBlock.tsx   # 音频块
│       ├── ChoiceBlock.tsx  # 选择题块
│       └── LinkBlock.tsx    # 链接块
```

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

启动后访问 http://localhost:5173 即可使用。

## 使用说明

1. 点击「新建卡包」创建一个卡包，自动进入并打开第一张卡片
2. 点击底部「+」按钮添加不同类型的内容块
3. 使用顶部工具栏对文字进行格式化
4. 拖拽手柄可对块进行排序
5. 底部翻页栏切换卡片，点击「+ 新建卡片」添加更多卡片
6. 点击「预览」切换到阅览模式，双面卡片可点击翻转
7. 预览模式下点击「导出」可保存为 PNG 图片或 PDF
8. 编辑内容会自动保存，无需手动操作
